#发现影响优先级
1ArkTS 运行时占主进程 RSS 58.5%，是最大内存消费者优化的首要目标高
2Swap 使用明显（主进程 56MB / GPU 12MB），系统内存有压力长期运行可能更严重高
3虚拟地址 47.5GB，但 98.5% 未分配物理页（Chromium partition_alloc）不影响实际内存，属正常行为信息
4GPU 进程 86% RSS 来自 shared_lib（驱动/渲染库）应用层面优化空间有限信息
5region_type 分类粗糙，ArkTS Heap 等归为 "other"影响分析精度中
6每次重启 PID 变化，需反复手动查找操作门槛高、易出错高
